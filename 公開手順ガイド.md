## ② GitHub Pages で無料公開する手順

### 必要なもの
- GitHubアカウント（無料）→ https://github.com

### 手順

1. **GitHubにログイン** → 右上「+」→「New repository」

2. **リポジトリを作成**
   - Repository name: `chotto-hima`
   - Public を選択
   - 「Create repository」をクリック

3. **ファイルをアップロード**
   - 「uploading an existing file」をクリック
   - `chotto-hima` フォルダの中身をすべてドラッグ&ドロップ
   - 「Commit changes」をクリック

4. **GitHub Pagesを有効にする**
   - リポジトリの「Settings」→「Pages」
   - Source: `Deploy from a branch`
   - Branch: `main` / `/(root)`
   - 「Save」をクリック

5. **数分後にアクセス確認**
   - `https://あなたのユーザー名.github.io/chotto-hima/` でアクセス可能

### 独自ドメインを設定する場合
- 「Settings」→「Pages」→「Custom domain」に `chotto-hima.com` を入力
- ドメイン会社のDNS設定で `CNAME` を `あなたのユーザー名.github.io` に向ける

## ③ Vercel で公開する（より簡単・おすすめ）

1. **Vercelアカウント作成** → https://vercel.com（GitHubアカウントでログイン可）
2. 「Add New Project」→「Upload」でフォルダをアップロード
3. 数秒でデプロイ完了・URLが発行される
4. 独自ドメインは「Settings」→「Domains」から追加

## ④ Google AdSense の設定

### 申請の流れ
1. **AdSenseアカウント作成** → https://adsense.google.com
2. サイトURLを登録して審査を申請
3. `<head>` タグ内に審査用コードを貼り付ける（各HTMLファイルに追加）
4. **審査通過後**（数日〜数週間）、広告コードが発行される

### 審査を通りやすくするためのポイント
- コンテンツを最低5〜10ページ以上用意する（追加のゲーム・占いを増やす）
- プライバシーポリシーページを設置済み ✅
- スマホ対応済み ✅
- 独自ドメインで運用していると有利

### 広告コードの貼り方（審査通過後）
各HTMLファイルの `<div class="ad-box">広告スペース</div>` を以下に置き換える：
```html
<ins class="adsbygoogle"
     style="display:block"
     data-ad-client="ca-pub-XXXXXXXXXXXXXXXX"
     data-ad-slot="XXXXXXXXXX"
     data-ad-format="auto"
     data-full-width-responsive="true"></ins>
<script>(adsbygoogle = window.adsbygoogle || []).push({});</script>
```

## ⑤ Google Search Console への登録（SEO）

Googleに「このサイト存在します！」と伝える作業。

1. https://search.google.com/search-console にアクセス
2. 「プロパティを追加」→ URLを入力
3. 所有権確認（HTMLファイルをアップロードするか、メタタグを貼る）
4. 「サイトマップ」→ `https://chotto-hima.com/sitemap.xml` を送信
